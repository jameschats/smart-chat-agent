Installation Steps

The user has to install the Recommendation System app from the market place.

Application Usage

The app provides recommendation to the agent when the agent chats with the visitor.

When the agent asks any question, the app provides 1 or 2 answers as recommendation.

If the answer is good, the agent can click on use icon button to send the answer as response to the agent.

If the answer needs modification, the agent can click on edit icon button, to edit the answer and send the answer as response to the agent.
